### Prerequisites

In order to run this project you need:

Ruby 3.0.1 or above
Rails 7.07 or above

### Setup

```git clone github.com/redmarverick/blog_app.git```


### Install

go to the folder of the repository, and run this line in the terminal to install dependences:

```bundle install```

### Usage
